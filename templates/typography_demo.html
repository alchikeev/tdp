Специальные эффекты

Градиентный текст

Используйте класс .text-gradient для создания градиентного текста.

Тень текста

Класс .text-shadow добавляет мягкую тень к тексту для лучшей читаемости.

Большая тень текста

Класс .text-shadow-lg создает более выраженную тень.

Инструкции по использованию

CSS переменные

Используйте CSS переменные для быстрого изменения шрифтов:

font-family: var(--font-primary);    /* Inter */
font-family: var(--font-secondary);  /* Poppins */
font-family: var(--font-narrow);     /* Montserrat/Raleway/Oswald - Kargert Alternative */
font-family: var(--font-condensed);  /* Roboto Condensed/Oswald */
font-family: var(--font-display);    /* Orbitron/Exo 2/Rajdhani - Hero Headlines */
font-family: var(--font-futuristic); /* Audiowide/Orbitron/Exo 2 */
font-family: var(--font-ultra-narrow); /* Space Grotesk/Rajdhani/Oswald - Ultra Narrow */
font-family: var(--font-mono-narrow);  /* Space Mono/JetBrains Mono/Fira Code */
font-family: var(--font-accent);     /* Playfair Display */
font-family: var(--font-mono);       /* SF Mono */
